# Mini Healthcare Support Web App

## Description
A simple concept-level healthcare support portal for NGOs.

## Features
- Patient Support Form
- Volunteer Registration
- Rule-based FAQ Chatbot (AI concept)

## AI / Automation Idea
A basic chatbot that answers common healthcare questions automatically.

## Tech Stack
- HTML
- CSS
- JavaScript

## Use Case
Small NGOs can collect patient requests, manage volunteers, and offer instant help.

## Deployment
Can be hosted on:
- Netlify
- Vercel
- Firebase Hosting
